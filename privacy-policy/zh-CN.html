Lenna 隐私政策

生效日期：2025年4月15日

最后更新：2025年4月15日

重要说明： 本隐私政策描述了 Lenna（"我们"、"本应用"）如何收集、使用、存储和分享信息，包括当您使用谷歌登录服务时的谷歌用户数据。

欢迎使用 Lenna，这是一款专为家庭娱乐设计的媒体播放器应用。我们致力于保护您的隐私，并对我们如何处理您的信息保持透明。

1. 我们收集的信息

1.1 谷歌用户数据（使用谷歌登录时）

当您选择使用谷歌账户登录时，我们会从您的谷歌账户访问并收集以下信息：

  • 基本资料信息： 您的姓名、电子邮箱地址和头像
  • 谷歌账户ID： 您谷歌账户的唯一标识符

1.2 应用数据

本应用可能在您的设备上本地存储以下信息：

  • 媒体服务器连接设置（Emby/Jellyfin 服务器 URL 和凭据）
  • 播放偏好和设置
  • 观看历史和进度（本地存储或通过 iCloud/Trakt.TV 同步）

2. 我们如何使用您的信息

2.1 谷歌用户数据使用

我们将您的谷歌用户数据用于以下目的：

  • 身份验证： 验证您的身份并提供对应用的安全访问
  • 用户体验： 通过显示您的姓名和头像来个性化您的体验
  • 账户管理： 将您的应用偏好和设置与您的账户关联

2.2 应用数据使用

您的应用数据用于：

  • 提供媒体流媒体和播放功能
  • 在设备间同步您的观看进度
  • 维护您的偏好和设置

3. 我们如何存储您的信息

通过谷歌登录获得的谷歌用户数据会被安全处理，不会永久存储在我们的服务器上。身份验证令牌按照谷歌的安全标准处理。

应用数据主要存储在您的设备上，可选择通过以下方式同步：

  • 苹果 iCloud（适用于 iOS 用户）
  • Trakt.TV（如果您选择连接您的账户）

4. 信息共享和披露

我们不会向第三方出售、交易或出租您的个人信息。我们仅在以下情况下共享您的信息：

  • 征得您的同意： 当您明确授权我们共享特定信息时
  • 服务提供商： 与帮助我们运营应用的可信第三方服务（如 iCloud 或 Trakt.TV），在严格保密协议下
  • 法律要求： 当法律要求或为保护我们的权利和用户安全时

5. 谷歌 API 服务用户数据政策合规

我们对从谷歌 API 接收信息的使用遵守谷歌 API 服务用户数据政策，包括限制使用要求。

我们承诺：

  • 仅将谷歌用户数据用于本隐私政策中披露的目的
  • 除了提供或改进面向用户的功能所必需外，不向第三方传输谷歌用户数据
  • 不使用谷歌用户数据投放广告
  • 除非我们获得您对特定消息的明确同意，否则不允许人工读取谷歌用户数据

6. 数据安全

我们实施适当的技术和组织措施来保护您的信息免受未经授权的访问、更改、披露或破坏。这包括：

  • 谷歌登录的安全身份验证协议
  • 传输和静态敏感数据的加密
  • 定期安全评估和更新

7. 您的权利和选择

您对您的信息拥有以下权利：

  • 访问权： 您可以请求访问我们拥有的关于您的信息
  • 更正权： 您可以请求更正不准确的信息
  • 删除权： 您可以请求删除您的账户和相关数据
  • 撤销访问权： 您可以随时通过谷歌账户设置撤销应用对您谷歌账户的访问权限

8. 儿童隐私

本应用不面向13岁以下儿童。我们不会故意收集13岁以下儿童的个人信息。如果我们发现收集了13岁以下儿童的个人信息，我们将立即采取措施删除此类信息。

9. 隐私政策变更

我们可能会不时更新本隐私政策以反映我们做法或适用法律的变化。当我们进行重大变更时，我们将：

  • 更新本政策顶部的"最后更新"日期
  • 如果我们有您的联系信息，通过应用或电子邮件通知用户
  • 在变更生效前为您提供审查变更的机会

10. 国际数据传输

您的信息可能会被传输到您所在国家以外的国家进行处理。我们确保此类传输符合适用的数据保护法律并实施适当的保障措施。

11. 联系我们

如果您对本隐私政策或我们的数据处理做法有任何问题、关切或请求，请联系我们：

电子邮箱： emby@outlook.my

回复时间： 我们将在30天内回复您的询问

本隐私政策自上述日期起生效，适用于 Lenna 应用的所有用户。
